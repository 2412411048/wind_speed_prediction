\section{引言}

\subsection{项目背景}
风速预测在可再生能源发电、气象预报、航空安全等领域具有重要的实际应用价值。准确的风速预测可以帮助风力发电厂优化发电计划，提高能源利用效率。本项目基于多高度传感器采集的风速及相关气象数据，利用机器学习方法构建风速预测模型。

\subsection{项目目标}
本项目的主要目标包括：
\begin{enumerate}
    \item 基于10米、50米、100米三个高度的传感器数据，构建风速预测模型；
    \item 完成单步预测和多步预测（1小时和16小时）任务；
    \item 对比至少3种不同原理的模型性能；
    \item 对模型预测结果进行可视化分析和评估。
\end{enumerate}

\subsection{报告结构}
本报告共分为以下几个部分：第二章介绍数据来源与预处理方法；第三章阐述特征工程过程；第四章描述模型构建与训练；第五章展示实验结果与分析；第六章总结与展望。

\section{数据描述与预处理}

\subsection{数据来源}
本项目数据来源于Hugging Face平台，包含三个不同高度（10米、50米、100米）的传感器数据集。每个数据集包含以下字段：
\begin{itemize}
    \item \textbf{时间戳（Timestamp）}：数据采集的具体时间
    \item \textbf{风速（Wind Speed）}：目标预测变量
    \item \textbf{风向（Wind Direction）}：风向角度
    \item \textbf{温度（Temperature）}：环境温度
    \item \textbf{气压（Pressure）}：大气压力
    \item \textbf{湿度（Humidity）}：空气湿度
\end{itemize}

数据总量约为10,573条，总大小约257KB。

\subsection{数据加载与合并}
三个高度的数据分别存储为Parquet格式，包含训练集、验证集和测试集三个文件。加载流程如下：

\begin{lstlisting}[language=Python, caption=数据加载与合并代码]
import pandas as pd

# 加载三个高度的数据
heights = ['10m', '50m', '100m']
dfs = {}

for height in heights:
    train_df = pd.read_parquet(f"data/{height}/train-00000-of-00001.parquet")
    val_df = pd.read_parquet(f"data/{height}/val-00000-of-00001.parquet")
    test_df = pd.read_parquet(f"data/{height}/test-00000-of-00001.parquet")
    df = pd.concat([train_df, val_df, test_df], ignore_index=True)
    df['timestamp'] = pd.to_datetime(df['timestamp'])
    df = df.sort_values('timestamp').reset_index(drop=True)
    dfs[height] = df

# 按时间戳合并三个高度
df_merged = dfs['10m'].merge(dfs['50m'], on='timestamp', suffixes=('_10m', '_50m'))
df_merged = df_merged.merge(dfs['100m'], on='timestamp')
\end{lstlisting}

\subsection{数据清洗}

\subsubsection{缺失值处理}
首先检查各列的缺失值情况，发现存在少量缺失值。采用线性插值法（Linear Interpolation）进行填充，该方法能够较好地保持时间序列的连续性：

\begin{lstlisting}[language=Python]
# 线性插值填充缺失值
df_merged = df_merged.interpolate(method='linear', limit_direction='both')
df_merged = df_merged.dropna()
\end{lstlisting}

\subsubsection{异常值处理}
采用3$\sigma$原则识别并处理异常值。对于每个数值型特征，计算其均值和标准差，将超出$[\mu - 3\sigma, \mu + 3\sigma]$范围的数据点视为异常值并予以删除：

\begin{lstlisting}[language=Python]
# 3σ原则处理异常值
numeric_cols = df_merged.select_dtypes(include=[np.number]).columns
for col in numeric_cols:
    mean = df_merged[col].mean()
    std = df_merged[col].std()
    df_merged = df_merged[(df_merged[col] > mean - 3*std) & 
                          (df_merged[col] < mean + 3*std)]
\end{lstlisting}

\section{特征工程}

\subsection{时间特征提取}
从时间戳中提取以下时间特征：
\begin{itemize}
    \item \textbf{小时（Hour）}：反映风速的日周期变化
    \item \textbf{星期几（Day of Week）}：反映风速的周变化模式
    \item \textbf{月份（Month）}：反映风速的季节性变化
    \item \textbf{是否周末（Is Weekend）}：二值特征，区分工作日与周末
\end{itemize}

\subsection{滞后特征构造}
对于时间序列预测，历史值对当前预测具有重要影响。构造以下滞后特征：
\begin{itemize}
    \item 前1小时风速（lag\_1h）
    \item 前3小时风速（lag\_3h）
    \item 前6小时风速（lag\_6h）
    \item 前12小时风速（lag\_12h）
\end{itemize}

\subsection{滑动窗口统计特征}
为捕捉风速的短期变化趋势，构造滑动窗口统计特征：
\begin{itemize}
    \item 过去3小时风速均值（rolling\_3h\_mean）
    \item 过去6小时风速均值（rolling\_6h\_mean）
    \item 过去3小时风速标准差（rolling\_3h\_std）
    \item 过去6小时风速标准差（rolling\_6h\_std）
\end{itemize}

\subsection{模型1：线性回归（Linear Regression）}
线性回归是最基础的预测模型，假设输入特征与目标变量之间存在线性关系：

\begin{equation}
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n + \varepsilon
\end{equation}

其中$\beta_i$为回归系数，$\varepsilon$为误差项。该模型训练速度快，可解释性强，适合作为基线模型。

\subsection{模型2：长短期记忆网络（LSTM）}
LSTM是一种特殊的循环神经网络（RNN），通过引入门控机制有效解决了传统RNN的梯度消失问题。其核心结构包括：

\begin{itemize}
    \item \textbf{遗忘门}：决定丢弃哪些历史信息
    \item \textbf{输入门}：决定存储哪些新信息
    \item \textbf{输出门}：决定输出哪些信息
\end{itemize}

本项目中LSTM模型采用2层堆叠结构，隐藏单元数为64，使用Dropout防止过拟合。

\subsection{模型3：Transformer}
Transformer模型基于自注意力机制（Self-Attention），能够捕捉序列中任意位置之间的依赖关系。其核心优势包括：

\begin{itemize}
    \item \textbf{并行计算}：相比RNN，训练速度更快
    \item \textbf{长距离依赖}：能够有效捕捉长期依赖关系
    \item \textbf{多头注意力}：从不同角度关注序列信息
\end{itemize}

本项目中Transformer模型采用2层编码器结构，4个注意力头，模型维度为64。

\section{实验结果与分析}

\subsection{评估指标}
采用以下四个指标评估模型性能：

\begin{enumerate}
    \item \textbf{均方误差（MSE）}：
    \begin{equation}
    MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2
    \end{equation}
    
    \item \textbf{均方根误差（RMSE）}：
    \begin{equation}
    RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}
    \end{equation}
    
    \item \textbf{平均绝对误差（MAE）}：
    \begin{equation}
    MAE = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|
    \end{equation}
    
    \item \textbf{决定系数（$R^2$）}：
    \begin{equation}
    R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}
    \end{equation}
\end{enumerate}

\subsection{结果讨论}
\begin{enumerate}
    \item \textbf{模型性能排序}：Transformer $>$ LSTM $>$ 线性回归，表明复杂的深度学习模型能够更好地捕捉风速序列的非线性特征。
    
    \item \textbf{预测难度分析}：多步预测（16小时）的误差显著大于单步预测和短时预测（1小时），这是因为预测时间越长，累积误差越大。
    
    \item \textbf{特征重要性}：通过特征相关性分析发现，滞后特征（尤其是lag\_1h和lag\_3h）与目标变量的相关性最高，说明风速具有较强的时间依赖性。
\end{enumerate}

\begin{enumerate}
